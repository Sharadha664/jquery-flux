1. Sizzle Engine

Sizzle is the selector engine behind jQuery that helps quickly identify DOM elements using CSS-like expressions.

2. DOM Selectors

DOM selectors are used in jQuery to target ids, classes, tags, and nested structures efficiently.

3. Event Delegation

Event delegation improves performance by attaching one event listener to a parent instead of many child elements.

4. AJAX Streams

AJAX lets pages load data dynamically without refreshing, creating faster and smoother user experiences.

5. Fade Effects

Fade effects make UI transitions smoother by gradually showing or hiding content.

6. Attr Control

Attribute control helps read, update, or remove HTML attributes like src, href, and title in jQuery.

7. CSS Variable Hooks

These are techniques used to connect jQuery with styling logic and update interface design dynamically.

8. Form Serialization

Serialization converts form fields into structured data that can be sent easily through AJAX requests.

9. Method Chaining

Method chaining allows multiple jQuery actions to be written in one clean sequence on the same element.

Closest traversal helps find the nearest matching parent element in the DOM tree.
